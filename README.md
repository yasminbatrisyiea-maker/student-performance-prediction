# Student Performance Prediction Using Machine Learning

## Overview
This project aims to predict student academic performance using machine learning techniques and predictive analytics.

## Objectives
- Analyze student academic data
- Identify factors affecting academic performance
- Build predictive models
- Support data-driven decision making

## Technologies Used
- Python
- Machine Learning
- Data Analytics
- Regression Analysis

Final Year Project (FYP)
